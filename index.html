pocketnes

pocketnes An NES emulator for GBA This is a mirror of the code taken from the archive.org mirror. It was last updated on July 1, 2013. The last version released was 9.98. While this is mostly for posterity's sake, there is a goal to implement the NES 2.0 iNES format. If you can help in any way, it would be greatly appreciated!
